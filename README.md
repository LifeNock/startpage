## System Modules

| **Core** | **Customization** |
| :--- | :--- |
| **Strict Mode** - Enforces HTTPS protocols on all shortcuts. | **Theme Engine** - Custom wallpapers, accent colors, and fonts. |
| **Weather Stealth** - Displays temperature while masking location data. | **Floating UI** - Draggable, OS-style settings window. |
| **Local Vault** - 100% LocalStorage. No databases, no tracking. | **Identity** - Upload custom Favicons and user aliases. |
| **Omnibox** - Smart search routing (Google, DDG, Brave, StartPage). | **Layout Control** - Switch sidebar alignment (Left/Right). |

## Installation

### Option A: GitHub Pages (Recommended)
1. Fork or Clone this repository.
2. Go to **Settings > Pages**.
3. Select `main` branch and `/root` folder.
4. Set the generated URL as your browser homepage.

### Option B: Local Host
```bash
git clone [https://github.com/LifeNock/startpage.git](https://github.com/LifeNock/startpage.git)
cd startpage
# Open index.html in your browser
